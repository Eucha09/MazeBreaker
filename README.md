# MazeBreaker – Source Code Repository

본 저장소는 Unity 기반 생존·액션 어드벤처 게임 MazeBreaker의 소스 코드 전용 저장소입니다.
리소스(모델, 텍스처, 사운드 등 대용량 파일)는 용량 문제로 제외되어 있으며,
프로젝트 구조, 핵심 시스템 코드, 개발 패턴 및 주요 기술 구현을 확인하기 위한 용도로 제공합니다.

### 📌 프로젝트 개요

MazeBreaker는 영화 Maze Runner에서 영감을 받아 제작된 동적으로 변화하는 미로에서의 생존·액션 어드벤처 게임입니다.

### 📌 개발 환경

- Unity 6 (6000.0.58f1)
- 플랫폼: Windows PC (빌드 불가)

### 📌 빌드 관련 안내

이 저장소는 리소스 미포함 상태이므로
게임 실행 및 빌드는 불가능합니다.
